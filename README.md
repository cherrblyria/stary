## ✨ Stary

> "Yet Another Start Page"

> 👉🏻 [Live Preview](https://cherrblyria.github.io/stary/)

### Notable Features

- Bookmarks
- Dark/Light Mode Support

### Customization

> Simply just fork this repo and modify the codebase

- Bookmarks: `bookmarks.json`
- Images: square image is recommended
  - Dark Mode: `assets/dark.jpg`
  - Light Mode: `assets/light.jpg`
- Fonts: `css/style.css`
- Anything else: `index.html`
